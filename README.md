# Kaast

Kaast is a desktop application that turns any web article into a ready-to-publish video podcast. Paste a URL, and Kaast handles the rest: AI generates the script, synthesizes the voice, selects matching stock footage, and assembles everything into an MP4. It runs natively on Windows, macOS, and Linux.

## Features

### AI-powered pipeline
- 🎙️ **Script generation** — Paste a URL, get a podcast script generated by Google Gemini
- 🎨 **Customizable style** — Define tone, format, and persona via a reusable style prompt
- 🗣️ **Voice synthesis** — Text-to-speech with Gemini TTS (natural-sounding narration)
- 🎬 **Scene analysis** — AI suggests video scenes with timestamps and search keywords
- 📹 **Stock footage** — Automatic download of matching videos from Pexels

### Editing & export
- ✂️ **Timeline editor** — Arrange audio and video clips on a multi-track timeline
- 🎞️ **Real-time preview** — Frame-perfect playback with Remotion Player
- ✨ **Transitions** — Fade, dissolve, wipe between video clips
- 📤 **MP4 export** — Configurable quality presets with real-time progress
- 📦 **Embedded FFmpeg** — No manual installation needed

### General
- 🌐 **Multilingual** — EN, FR, DE, ES, IT, PT, NL, PL (auto-detected)
- 💾 **Project management** — Create, reopen, and auto-save projects

## Technologies

- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Rust + Tauri 2.0
- **Preview**: Remotion Player (frame-perfect audio/video playback)
- **Video Processing**: FFmpeg / FFprobe (bundled as Tauri sidecar)
- **AI**: Google Gemini API (script generation, voice synthesis, scene analysis)
- **Stock Footage**: Pexels API
- **i18n**: react-i18next with i18next-browser-languagedetector
- **UI Theme**: DaVinci Resolve-inspired dark theme with orange accent

## Supported Platforms

- ✅ Windows 10/11 (x86_64)
- ✅ macOS (ARM aarch64 + Intel x86_64)
- ✅ Linux (x86_64) — AppImage / DEB / RPM

## Project Structure

```
src/                          # React frontend
├── components/               # UI components (EditPage, ScenesPage, ScriptEditor, ...)
├── hooks/                    # Custom hooks (useRemotionSync, useMediaDuration)
├── remotion/                 # Remotion compositions (PodcastComposition, VideoClipSequence)
├── i18n/locales/             # Translation files (en, fr, de, es, it, pt, nl, pl)
├── utils/                    # Shared utilities (tauri invocation, project helpers)
├── types/                    # TypeScript type definitions
└── styles.css                # Global styles (DaVinci Resolve theme)

src-tauri/src/                # Rust backend
├── main.rs                   # Tauri bootstrap
├── gemini.rs                 # Gemini API (script generation, TTS, scene analysis)
├── ffmpeg.rs                 # FFmpeg commands (cut, merge, transition, export, thumbnail, proxy)
├── pexels.rs                 # Pexels API (stock video search & download)
├── files.rs                  # File I/O utilities (media file reading)
└── sidecar.rs                # Sidecar binary resolution (FFmpeg/FFprobe)

scripts/                      # Build and setup scripts
└── download-ffmpeg.sh        # Download FFmpeg binaries for current platform
```

## Prerequisites (development)

- [Node.js](https://nodejs.org/) (v18 or later)
- [Rust](https://www.rust-lang.org/tools/install)

### Installing Rust

**Windows**:
1. Download and run [rustup-init.exe](https://win.rustup.rs/)
2. Follow the installer instructions
3. Restart your terminal

**macOS / Linux**:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Installation

```bash
npm install
npm run download-ffmpeg
```

## Development

```bash
npm run tauri dev
```

> **Note**: Do not use `npm run dev` alone — it only starts the frontend without the Rust backend.

The first build may take several minutes as Rust compiles all dependencies.

## Build

```bash
npm run tauri build
```

The executable will be generated in `src-tauri/target/release/`.
